# 路径

##### 1.目录文件夹和根目录。

**目录文件夹**：就是普通文件夹

**根目录**：打开目录文件夹的第一层。



##### 2.相对路径

以引用文件所在位置为参考基础，而建立出的目录路径。

| 相对路径分类 | 符号 |
| ------------ | ---- |
| 同级路径     |      |
| 下一级路径   | /    |
| 上一级路径   | ../  |

例：

```html
<img src="nb.jpg /">
<img src="images/nb.jpg" />
<img src="../nb.jpg" />
```

##### 3.绝对路径

目录下的绝对位置。

例：

```html
<img src="D:\html\nb.jpg /">
<img src="http://www.nb.com"//nb//nb.jpg />
```
